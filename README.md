# AI_playing_flappy_bird
Using Deep-Q learning to AI to play game flappy bird.


## Requirements : 
1. tensorflow
2. Numpy
3. pygame

### How To Use :

#### Train New model : 
```
python3 train.py
```

#### Play using trained model : 
``` python3 play.py ```

#### Play the game : 
``` python3 game.py ```
